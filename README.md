# Lox in Rust
This is an implementation of the Lox language written in Rust. Lox is an
object-oriented scripting language described and implemented in the book
[Crafting Interpreters](https://craftinginterpreters.com/) by
[Bob Nystrom](https://github.com/munificent).

I have already written a [Lox interpreter in C](https://github.com/krobbi/lox)
which closely follows the book, but this is an attempt to write a less guided
implementation with my own design choices.

*Crafting Interpreters is a copyright of Bob Nystrom.*
